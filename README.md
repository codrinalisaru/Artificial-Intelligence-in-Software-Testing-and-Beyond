# Artificial-Intelligence-in-Software-Testing-and-Beyond
Replication package: Artificial Intelligence in Software Testing and Beyond: A Review of Current Practices and Emerging Challenges
End-to-end reproduction of data extraction, cleaning, analysis, figures/tables, and statistics for the 2015–2025 corpus (72 papers) across themes T1 (AI-for-Testing), T2 (Testing/Validation of AI), T3 (“Beyond” topics with testing implications).
